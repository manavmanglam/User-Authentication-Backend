# User-Authentication-Backend
🔐 **Implemented Secure JWT Authentication in a MERN Stack Application**

I'm excited to share another milestone in my Full Stack Development journey—successfully implementing **JWT-based User Authentication** in a MERN Stack application.

This project focused on building a secure authentication system that enables users to register, log in, and securely access protected resources using **JSON Web Tokens (JWT)**.

### 🚀 Key Features

✅ User Registration & Login
✅ Secure Password Hashing with **bcrypt**
✅ JWT Token Generation & Verification
✅ Protected Backend Routes using Authentication Middleware
✅ Persistent User Sessions with Token Storage in **localStorage**
✅ Authentication Flow between React Frontend and Express Backend

### 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Mongoose
* JSON Web Token (JWT)
* bcrypt
* JavaScript (ES6+)

### 📚 What I Learned

* Implementing secure user authentication workflows
* Password encryption and best security practices
* Token-based authentication and authorization using JWT
* Protecting API endpoints with middleware
* Managing authentication state in React
* Integrating frontend and backend for secure user sessions
